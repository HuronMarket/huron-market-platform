# Project Basket

## Mission
Increase basket size while making Huron Market the easiest place to shop for tonight.

## Core Principles
- Basket First
- Alcohol First
- Built for Huron Market
- Data Driven
- AI Assisted

## Every feature must:
- Increase revenue,
- Reduce labor, or
- Improve the customer experience.

If it doesn't accomplish one of those goals, it doesn't belong in Project Basket.

## Priority Order
1. Sell more alcohol.
2. Increase basket size.
3. Make Huron Market easier to shop.
4. Save employees time.
5. Generate marketing automatically.

## Motto
"Every click should lead to another item in the basket."
